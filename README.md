# موقع شركة — قالب عربي RTL بسيط

قالب صفحة هبوط/تعريفي يدعم العربية (RTL) بدون أُطر عمل، جاهز للتعديل السريع.

## ماذا يحتوي؟
- صفحة `index.html` منظمة للأقسام الأساسية: الرئيسية، من نحن، خدماتنا، أعمالنا، آراء العملاء، تواصل معنا.
- تنسيقات حديثة في `css/style.css` مع دعم الوضع الداكن/الفاتح.
- تفاعلات بسيطة في `js/main.js` (قائمة الجوال، تمرير سلس، تحقق نموذج، زر أعلى الصفحة).

## البدء السريع
يمكن فتح الملف مباشرة في المتصفح:
1. افتح المجلد في VS Code.
2. افتح الملف `index.html` واضغط زر Open with Live Server (إن كان لديك إضافة Live Server)، أو انقر عليه مرتين لفتحه مباشرة في المتصفح.

بديل خفيف (اختياري) لتشغيل خادم محلي إن كان لديك Node.js:
```powershell
# تثبيت خادم بسيط مؤقتاً (مرة واحدة)
npm -g i serve
# تشغيل في الجذر
serve -p 5500 .
```
ثم افتح: http://localhost:5500

> ملاحظة: الأيقونات والصور المستخدمة هنا تجريبية عبر placehold.co — استبدلها بصورك.

## التخصيص
- اسم الشركة والشعار: ابحث في `index.html` عن نص "اسم شركتك" واستبدله باسم شركتك.
- الألوان: غيّر متغيرات CSS أعلى `css/style.css` (مثل `--primary`).
- الخدمات: حرّر بطاقات الخدمات داخل قسم "خدماتنا" في `index.html`.
- الأعمال: استبدل صور المعرض داخل قسم "أعمالنا" بروابط صورك أو ضع ملفاتك داخل `assets/img/` واستخدم المسارات.
- وسائل التواصل: حدث روابط القسم السفلي (واتساب/تويتر/لينكدإن).

## إرسال نموذج التواصل
النموذج حالياً يعرض رسالة نجاح فقط. لربطه بخادم:
- أبسط خيار: استخدم خدمة خارجية مثل Formspree أو Getform وأضف `action` و`method="POST"` للنموذج.
- أو أنشئ واجهة برمجية لديك (Node/Express أو أي تقنية) ثم أرسل البيانات عبر `fetch` في `js/main.js`.
